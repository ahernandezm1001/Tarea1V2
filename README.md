# Spring_Boot_Ejercicio1_Hola_Spring
 Primer ejercicio de creación de un proyecto con Spring Boot, este devuelve un recurso HTML con un encabezado "Hola Spring" mediante un controlador que redirige a la URL localhost:8080/Hola_Spring

# Versión de java
OpenJDK Runtime Environment Corretto-23.0.2.7.1

# Versión de Maven
Apache Maven 3.9.9

# Dependencias utilizadas
Spring Boot Starter Web → Para crear una API REST.
Thymeleaf → Para renderizar HTML dinámico.
Bootstrap → Para estilos responsive.

# Instrucciones de ejecución
Primeramente asegurese de tener instalado:
    Java JDK 17+
    Apache Maven
    
Clone el repositorio con "git clone https://github.com/ahernandezm1001/Tarea1V2"
Compile el proyecto con "mvn spring-boot:run"
Finalmente abra un navegador y acceda a "http://localhost:8080/Hola_Spring"

# Nota adicional
Si requiere cambiar el puerto (8080 por defecto) hagalo en el archivo "application.properties" ubicado en "src/resources"
